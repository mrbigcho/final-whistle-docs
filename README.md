## 주요 기능 요약

- **클럽**: 같은 커뮤니티 사람들과 클럽을 만들어 대회와 리그를 함께 운영
- **대회 (토너먼트)**: 단일 탈락, 더블 탈락, 조별 리그 등 다양한 형식 지원
- **대진표**: 시각적인 토너먼트 대진표와 조별 리그 결과 확인
- **경기 결과 등록**: 플레이어가 직접 경기 결과를 등록하고 상대방이 확인
- **판타지 리그**: 선수 드래프트, 경매, 트레이드, 자유선수 입찰
- **실시간 경매**: 팀 예산 내에서 선수 경매 진행

## 지원 환경

Final Whistle은 **웹 브라우저** 기반으로 동작하며, 화면 크기에 따라 레이아웃이 자동으로 변경됩니다.

- **모바일** (600px 미만): 하단 탭 바
- **태블릿** (600px~1200px): 좌측 사이드 레일
- **데스크탑** (1200px 이상): 좌측 사이드 드로어
