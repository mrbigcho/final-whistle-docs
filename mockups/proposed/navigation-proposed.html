GoRouter

      ├── Branch[0] /home

      │          └── /competitions/:id // TabBar: 대진표/경기/참가자/순위표 (v1)
      ├── Branch[2] /clubs
      │          └── ShellRoute (ClubShell) // 클럽 컨텍스트 래퍼
      │               └── StatefulShellRoute.indexedStack // [L2] 클럽 BottomNav
      │                      ├── Branch[0] /clubs/:id/home
      │                      ├── Branch[1] /clubs/:id/competitions
      │                      │          └── /clubs/:id/competitions/:cid
      │                      ├── Branch[2] /clubs/:id/members
      │                      └── Branch[3] /clubs/:id/settings // Admin만 노출
      └── Branch[3] /my
개선 효과
✅ 클럽 컨텍스트 명확화
클럽 내 탐색 시 항상 클럽 전용 BottomNav가 표시되어 사용자가 현재 위치를 직관적으로 파악할 수 있다.
✅ 역할 기반 UI 분기
Admin에게만 설정 탭을 노출하여 일반 멤버에게 불필요한 메뉴를 제거하고, 관리자의 작업 효율을 높인다.
✅ Back 네비게이션 일관성
클럽 BottomNav 루트에서 Back → 클럽 목록, 딥링크 진입 시에도 클럽 컨텍스트 유지로 예측 가능한 동작을 보장한다.
✅ 대회 탭 확장 기반 마련
Phase 1에서 대회 상세 TabBar를 4탭으로 확장하고, Phase 2에서 독립 BottomNav로 전환하는 점진적 개선이 가능하다.
