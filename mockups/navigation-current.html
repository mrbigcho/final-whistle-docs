  └── StatefulShellRoute.indexedStack // 메인 BottomNav (4탭)
      ├── Branch[0] /home
      ├── Branch[1] /competitions
      │          └── /competitions/:id // TabBar: 대진표/순위표

      │          └── /clubs/:id  ← 내부 TabBar (대회/리그/멤버) — 클럽 컨텍스트만 존재

      └── Branch[3] /my
발견된 문제점
⚠ 문제 1 · 클럽 컨텍스트 손실
클럽 상세에서 대회 상세로 이동하면 BottomNav의 활성 탭이 클럽→대회로 변경된다. 사용자는 자신이 클럽 내 대회를 보는지, 전체 대회를 보는지 구분하기 어렵다.
⚠ 문제 2 · 관리 기능 접근성 저하
설정은 AppBar ⋮ 메뉴 안에 숨어 있어 관리자가 자주 사용하는 기능(멤버 초대, 공개 설정)에 대한 접근이 불편하다. 역할 기반 UI 분기도 없다.
⚠ 문제 3 · 대회 상세 탭 확장 한계
현재 대진표/순위표 두 탭만 있으며, 경기 일정·참가자 등 기능 추가 시 TabBar 레이블이 좁아진다. 클럽 내 대회와 전체 대회의 구조 불일치 문제도 발생 가능하다.
⚠ 문제 4 · Back 네비게이션 불명확
클럽 내 대회 → 대회 상세에서 Back을 누르면 클럽 상세로 돌아가야 하지만, 현재 구조에서는 이전 스택 상태에 따라 동작이 불일치할 수 있다.
