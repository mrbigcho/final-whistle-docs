# 시작하기

## 접속

웹 브라우저에서 **https://fw.dongne.dev/** 에 접속합니다.

---

## 회원가입

1. 접속 후 로그인 화면이 나타나면 상단 탭에서 **회원가입**을 선택합니다.
2. **이메일 주소**를 입력합니다.
3. **비밀번호**를 입력합니다. (6자 이상)
4. **회원가입** 버튼을 클릭합니다.
5. 가입이 완료되면 자동으로 로그인되며 홈 화면으로 이동합니다.

> **안내:** 회원가입 후 프로필이 자동으로 생성됩니다. 닉네임 등 추가 정보는 [프로필 수정](profile.md)에서 설정할 수 있습니다.

---

## 로그인

1. 접속 후 로그인 화면에서 **로그인** 탭을 선택합니다.
2. 가입 시 사용한 **이메일 주소**와 **비밀번호**를 입력합니다.
3. **로그인** 버튼을 클릭합니다.

> **오류 발생 시:** 이메일 또는 비밀번호가 틀렸을 경우 오류 메시지가 표시됩니다. 입력 내용을 확인한 후 다시 시도하세요.

---

## 기본 화면 구성

로그인 후 홈 화면이 표시됩니다.

### 하단 탭 (모바일) / 좌측 메뉴 (태블릿·데스크탑)

| 메뉴 | 설명 |
|------|------|
| **홈** | 대시보드, 빠른 접근 버튼 |
| **대회** | 토너먼트 대회 목록 및 관리 |
| **클럽** | 클럽 목록 및 관리 |
| **MY** | 내 계정, 프로필 설정 |

### 홈 화면

- **대회 보러 가기**: 대회 목록 화면으로 이동
- **클럽 보러 가기**: 클럽 목록 화면으로 이동
- 참가 중인 대회 및 클럽 소식 표시 (추후 지원 예정)

---

## 로그아웃

1. 하단 탭(또는 좌측 메뉴)에서 **MY**를 선택합니다.
2. **로그아웃** 버튼을 클릭합니다.
3. 로그인 화면으로 이동합니다.
