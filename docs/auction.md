# 경매

경매 시스템은 판타지 리그에서 선수를 팀에 배정하는 방식입니다. 각 팀은 정해진 예산 내에서 원하는 선수에 입찰하여 팀을 구성합니다.

---

## 경매 접근 방법

1. 리그 상세 화면에서 **경매** 버튼을 클릭합니다.
2. 경매 화면으로 이동합니다.

---

## 경매 상태

| 상태 | 설명 |
|------|------|
| **대기 중 (Pending)** | 관리자가 경매를 시작하기 전 상태 |
| **진행 중 (LIVE)** | 경매가 활성화된 상태 |
| **완료 (Completed)** | 모든 선수 경매가 종료된 상태 |

경매 상태는 화면 상단 배지로 표시됩니다.

---

## 경매 대기 상태

경매가 아직 시작되지 않으면 대기 배너가 표시됩니다.

- 관리자가 경매를 시작할 때까지 기다립니다.
- 새로고침하거나 잠시 후 다시 확인하세요.

---

## 경매 진행 중

경매가 시작되면 다음 화면이 표시됩니다.

### 현재 선수 카드

- **선수 이름**, **포지션**, **오버롤(OVR)** 정보
- **현재 최고 입찰가**: 현재까지의 최고 금액
- **현재 입찰자**: 가장 높은 입찰을 한 팀 이름
- **남은 시간**: 입찰 마감까지 남은 시간 (카운트다운)

### 입찰 방법

1. **입찰 금액 입력**: 현재 최고 입찰가보다 높은 금액을 입력합니다.
2. **입찰하기** 버튼을 클릭합니다.
3. 입찰이 성공하면 현재 입찰자가 내 팀으로 변경됩니다.

> **주의:** 입찰 금액은 현재 최고 입찰가보다 높아야 합니다.

### 내 예산 카드

- **남은 예산**: 현재 사용 가능한 예산
- **현재 로스터 수**: 내 팀에 영입된 선수 수

> **팁:** 남은 예산과 로스터 크기를 고려하여 전략적으로 입찰하세요. 예산이 부족하면 이후 선수를 영입할 수 없습니다.

### 팀 예산 현황 표

모든 팀의 예산 현황을 한눈에 확인할 수 있습니다.

| 항목 | 설명 |
|------|------|
| **팀명 (사용자)** | 팀 소유자의 닉네임 |
| **남은 예산** | 해당 팀의 현재 남은 예산 |
| **로스터 수** | 현재 영입된 선수 수 |

---

## 경매 완료

모든 선수의 경매가 끝나면 완료 배너가 표시됩니다.

- 최종 결과 확인 가능
- 각 선수의 낙찰 팀과 낙찰 금액 확인 가능

---

## 관리자 패널 (관리자 전용)

리그 관리자에게는 추가 컨트롤 버튼이 표시됩니다.

| 버튼 | 기능 |
|------|------|
| **경매 시작** | 경매를 시작합니다 (대기 중 상태에서만 가능) |
| **다음 선수** | 현재 선수 경매가 끝난 후 다음 선수로 넘어갑니다 |
| **경매 완료** | 모든 경매를 종료합니다 |

---

## 경매 전략 팁

- **예산 배분**: 핵심 선수에 많은 예산을 투자하고, 나머지 선수는 저렴하게 채우는 전략이 효과적입니다.
- **경쟁 관찰**: 다른 팀의 남은 예산을 확인하여 경쟁 강도를 파악하세요.
- **마지막 라운드**: 다른 팀의 예산이 소진될 때 좋은 선수를 저렴하게 영입할 수 있습니다.
- **로스터 크기 주의**: 로스터가 가득 차면 더 이상 선수를 영입할 수 없습니다.
