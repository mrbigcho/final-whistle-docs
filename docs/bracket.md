# 대진표

대진표 화면에서는 토너먼트의 전체 대진 구조를 시각적으로 확인하고, 각 경기 결과를 바로 등록할 수 있습니다.

---

## 대진표 접근 방법

1. **대회 목록**에서 원하는 대회를 선택합니다.
2. 대회 상세 화면 하단 탭에서 **대진표**를 선택합니다.

---

## 단일 탈락 / 더블 탈락 형식

### 화면 구성

- 대진표가 트리(Tree) 형태로 시각적으로 표시됩니다.
- 왼쪽에서 오른쪽으로 라운드가 진행됩니다.
- 각 매치 카드에는 다음 정보가 표시됩니다.

| 요소 | 설명 |
|------|------|
| **선수 이름** | 참가자 1 vs 참가자 2 |
| **점수** | 현재 경기 점수 |
| **우승자 표시** | 우승자 옆에 트로피 아이콘 표시 |
| **TBD** | 아직 결정되지 않은 대진 |

### 화면 조작

- **확대/축소**: 핀치 줌 또는 마우스 휠로 대진표를 확대·축소할 수 있습니다.
- **이동 (Pan)**: 드래그로 대진표를 이동할 수 있습니다.

### 경기 클릭

매치 카드를 클릭하면 해당 경기의 상세 화면으로 이동합니다.

- 경기가 아직 진행 중이면 **결과 등록** 버튼이 표시됩니다.
- 경기 결과 등록 방법은 [경기 결과](match.md) 문서를 참고하세요.

---

## 조별 리그 (Round Robin) 형식

### 화면 구성

- **라운드 선택**: 상단 드롭다운 메뉴에서 라운드를 선택합니다.
- **라운드 이동**: 이전/다음 버튼으로 라운드를 순서대로 이동합니다.
- 선택한 라운드의 모든 경기가 카드 목록으로 표시됩니다.

### 조별 경기 카드

각 경기 카드에는 다음이 표시됩니다.

- 참가자 1 vs 참가자 2
- 경기 결과 (점수, 우승자)
- 경기 상태 (대기 중 / 진행 중 / 완료)

---

## 경기 상태별 표시

| 상태 | 표시 방식 |
|------|-----------|
| **대기 중** | 흐린 텍스트, 결과 없음 |
| **진행 중** | 강조 표시 |
| **완료** | 우승자 강조, 트로피 아이콘 |
| **TBD** | 이름 자리에 "TBD" 표시 |

---

## 팁

- 대진표가 너무 크면 축소해서 전체 구조를 한눈에 파악하세요.
- 특정 경기 결과를 등록하려면 해당 매치 카드를 직접 클릭하세요.
- 조별 리그에서는 라운드 선택으로 특정 주차 경기만 확인할 수 있습니다.
