# 대회

대회는 토너먼트 방식으로 운영되는 경기 이벤트입니다. 다양한 대진 형식을 지원하며, 참가자 모집부터 결과 집계까지 관리할 수 있습니다.

---

## 대회 목록 보기

1. 하단 탭(또는 좌측 메뉴)에서 **대회**를 선택합니다.
2. 참가 중이거나 관련된 대회 목록이 표시됩니다.

### 상태 필터

상단 필터 탭으로 대회를 상태별로 필터링할 수 있습니다.

| 필터 | 설명 |
|------|------|
| **전체** | 모든 대회 표시 |
| **모집 중** | 참가자를 모집 중인 대회 |
| **진행 중** | 현재 경기가 진행 중인 대회 |
| **완료** | 종료된 대회 |

---

## 대회 만들기

1. 대회 목록 화면에서 우하단 **+** 버튼을 클릭합니다.
2. **대회 만들기** 대화상자에서 정보를 입력합니다.

| 항목 | 설명 |
|------|------|
| **대회 이름** | 필수 입력 |
| **최대 참가 인원** | 선택 입력 (설정하지 않으면 인원 제한 없음) |

3. **만들기** 버튼을 클릭합니다.

> **팁:** 최대 참가 인원을 설정하는 경우, 대진 형식에 따라 2의 배수(4, 8, 16, 32, 64)로 설정하는 것이 권장됩니다. 설정하지 않으면 제한 없이 참가자를 받을 수 있습니다.

---

## 대회 상세 보기

대회 목록에서 대회를 클릭하면 상세 화면으로 이동하며, 화면 하단에 대회 전용 네비게이션이 표시됩니다.

### 하단 탭 구성

| 탭 | 내용 |
|----|------|
| **대진표** | 토너먼트 대진 구조 시각화 |
| **경기** | 대회 내 모든 경기 목록 |
| **참가자** | 참가자 요약 카드 (탭하면 전체 목록) |
| **순위표** | 스테이지별 순위 및 리더보드 |
| **정보** | 대회 메타데이터 및 설정 |

---

## 대회 진행 상태

| 상태 | 설명 |
|------|------|
| **모집 중 (Open)** | 참가자를 받고 있는 단계 |
| **진행 중 (In Progress)** | 경기가 시작된 단계 |
| **완료 (Completed)** | 모든 경기가 종료된 단계 |
| **취소 (Cancelled)** | 대회가 취소된 상태 |

### 상태 전환

대회 관리자는 **정보 탭**에서 대회 상태를 수동으로 전환할 수 있습니다.

---

## 대진 형식

Final Whistle은 다양한 대진 형식을 지원합니다.

| 형식 | 설명 |
|------|------|
| **단일 탈락 (Single Elimination)** | 한 번 지면 탈락하는 토너먼트 |
| **더블 탈락 (Double Elimination)** | 두 번 져야 탈락하는 토너먼트 |
| **조별 리그 (Round Robin)** | 모든 참가자가 서로 한 번씩 경기 |
| **조별 예선 + 토너먼트 (Group Stage)** | 조별 리그 후 토너먼트 방식 |

---

## 참가자 관리

**참가자 탭**에서 참가자 현황을 확인하고 관리합니다.

### 참가자 요약 카드

참가자 탭에 진입하면 현재 참가자 수와 간략한 현황을 보여주는 요약 카드가 표시됩니다.

- 카드를 탭하면 **참가자 전체보기** 화면으로 이동합니다.

### 참가자 전체보기

전체보기 화면에서 모든 참가자 목록과 관리 기능을 사용할 수 있습니다.

| 기능 | 설명 |
|------|------|
| **멤버 초대** | 클럽 멤버를 대회에 직접 참가시키기 (관리자) |
| **봇 추가** | 빈 자리를 채우기 위한 봇 참가자 추가 (관리자) |
| **참가자 삭제** | 참가자를 대회에서 제거 (관리자) |

> **팁:** 클럽 owner는 클럽 멤버 목록에서 원하는 멤버를 선택해 대회에 바로 참가시킬 수 있습니다.

---

## 대회 정보 수정

대회 관리자만 수정할 수 있습니다.

1. 대회 상세 화면의 **정보 탭**으로 이동합니다.
2. 대회 이름 등 정보를 변경합니다.
3. **저장** 버튼을 클릭합니다.

---

## 대진표 보기

대회 상세 화면의 **대진표 탭**을 선택하면 시각적인 대진표를 확인할 수 있습니다.

자세한 내용은 [대진표](bracket.md) 문서를 참고하세요.

---

## 주의사항

- 대회가 시작된 후에는 참가 인원을 변경하기 어려울 수 있습니다.
- 봇 참가자가 포함된 경기는 결과를 수동으로 입력해야 할 수 있습니다.
