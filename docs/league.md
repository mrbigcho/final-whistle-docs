# 리그 (판타지 리그)

판타지 리그는 가상의 선수를 드래프트하고, 팀을 구성하여 운영하는 리그 시스템입니다. 경매를 통해 선수를 영입하고, 트레이드와 자유선수 입찰로 팀을 강화합니다.

---

## 리그 접근 방법

리그는 클럽 내에서 생성됩니다.

1. **클럽** 메뉴에서 원하는 클럽을 선택합니다.
2. 클럽 상세 화면의 **홈 탭** 또는 **대회 탭**에서 연결된 리그를 확인합니다.
3. 원하는 리그를 클릭합니다.

---

## 리그 진행 단계

리그는 순서대로 다음 단계를 거칩니다.

| 단계 | 설명 |
|------|------|
| **드래프트 (Draft)** | 리그 준비 단계, 선수 추가 가능 |
| **경매 (Auction)** | 선수를 경매로 팀에 배정하는 단계 |
| **시즌 중 (In Season)** | 리그 경기 진행 중 |
| **플레이오프 (Playoff)** | 플레이오프 진행 중 |
| **완료 (Completed)** | 리그 종료 |

---

## 리그 상세 화면

### 탭 구성

| 탭 | 내용 |
|----|------|
| **선수 목록** | 리그에 등록된 전체 선수 |
| **팀 관리** | 내 팀 관리 (준비 중) |

### 리그 헤더 정보

- 리그 이름 및 시즌
- 현재 상태
- 팀당 예산
- 로스터 크기 (팀당 최대 선수 수)
- 멤버 수 및 개설자

---

## 선수 목록

**선수 목록 탭**에서 리그에 등록된 모든 선수를 확인합니다.

| 항목 | 설명 |
|------|------|
| **선수 이름** | 선수의 이름 |
| **포지션** | 선수 포지션 (있는 경우) |
| **오버롤 (OVR)** | 선수 능력치 (1~100, 색상으로 등급 표시) |
| **소속 팀** | 경매로 영입된 팀 이름 |
| **영입 금액** | 경매에서 낙찰된 가격 |

### 오버롤 등급 색상

| 등급 | 색상 |
|------|------|
| 90 이상 | 금색 (최상급) |
| 80~89 | 초록색 (상급) |
| 70~79 | 파란색 (중상급) |
| 그 이하 | 기본색 |

---

## 선수 추가 (관리자)

리그 관리자는 드래프트 단계에서 선수를 추가할 수 있습니다.

1. 리그 상세 화면에서 **선수 추가** 버튼(+ 아이콘)을 클릭합니다.
2. 선수 정보를 입력합니다.

| 항목 | 설명 |
|------|------|
| **선수 이름** | 필수 입력 |
| **포지션** | 선택 입력 |
| **오버롤** | 1~100 사이 숫자 |

3. **추가** 버튼을 클릭합니다.

---

## 경매 시작

선수가 모두 추가되면 경매를 시작합니다.

1. 리그 상세 화면에서 **경매** 버튼을 클릭합니다.
2. 경매 화면으로 이동합니다.

자세한 경매 방법은 [경매](auction.md) 문서를 참고하세요.

---

## 이적 시장

경매 이후 팀 구성을 조정하려면 이적 시장을 활용합니다.

리그 화면에서 **이적 시장** 버튼을 클릭하거나, 직접 이적 시장 화면으로 이동합니다.

자세한 내용은 [이적 시장](transfer.md) 문서를 참고하세요.

---

## 주의사항

- 경매는 리그 관리자가 시작해야 진행됩니다.
- 드래프트 단계에서만 선수를 추가할 수 있습니다.
- 팀 관리 기능은 현재 개발 중입니다.
