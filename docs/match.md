# 경기 결과 등록

경기 결과는 참가자가 직접 등록하고, 상대방이 이를 확인(컨펌)하는 방식으로 처리됩니다. 결과에 이의가 있을 경우 분쟁(Dispute)을 제기할 수 있습니다.

---

## 경기 상세 화면 접근 방법

- **대진표**에서 매치 카드를 클릭합니다.
- 또는 결과 등록 알림에서 직접 이동합니다.

---

## 경기 정보 확인

경기 상세 화면에서 다음 정보를 확인할 수 있습니다.

| 항목 | 설명 |
|------|------|
| **참가자** | 참가자 1 vs 참가자 2 |
| **경기 상태** | 대기 중 / 진행 중 / 완료 / 분쟁 |
| **예정 시간** | 경기 예정 일시 |
| **완료 시간** | 경기 완료 일시 |

---

## 경기 결과 등록

경기가 종료된 후 결과를 등록합니다.

1. 경기 상세 화면에서 **결과 등록** 버튼을 클릭합니다.
2. **승자**를 선택합니다. (라디오 버튼으로 참가자 중 한 명 선택)
3. **점수**를 입력합니다. (선택 사항, 예: `3-1`)
4. **등록** 버튼을 클릭합니다.

> **안내:** 결과를 등록하면 상대방에게 확인 요청이 전송됩니다.

---

## 결과 확인 (컨펌)

상대방이 결과를 등록하면 내가 확인해야 합니다.

1. 경기 상세 화면에 **결과 확인 요청** 메시지가 표시됩니다.
2. 등록된 결과가 올바르면 **확인** 버튼을 클릭합니다.
3. 결과가 잘못되었으면 **분쟁** 버튼을 클릭합니다.

---

## 결과 확인 상태

| 상태 | 설명 |
|------|------|
| **대기 중 (Pending)** | 아직 결과가 등록되지 않은 상태 |
| **확인 대기 중 (Awaiting Confirmation)** | 한 쪽이 결과를 등록했으나 상대방이 아직 확인하지 않은 상태 |
| **확인됨 (Confirmed)** | 양측이 결과에 동의한 상태 |
| **자동 확인 (Auto-Confirmed)** | 일정 시간 내에 응답이 없어 자동으로 확인된 상태 |
| **분쟁 (Disputed)** | 양측의 결과가 다른 상태 |

---

## 분쟁 처리

결과에 동의하지 않을 경우:

1. 경기 상세 화면에서 **분쟁** 버튼을 클릭합니다.
2. 경기 상태가 **분쟁(Disputed)**으로 변경됩니다.
3. 대회 관리자가 분쟁을 검토하고 최종 결과를 결정합니다.

---

## 경기 상태

| 상태 | 설명 |
|------|------|
| **대기 중 (Pending)** | 경기가 예정되어 있으나 아직 시작되지 않음 |
| **진행 중 (In Progress)** | 경기가 현재 진행 중 |
| **완료 (Completed)** | 경기 결과가 확정됨 |
| **분쟁 (Disputed)** | 결과에 이의가 제기된 상태 |

---

## 주의사항

- 경기 결과는 반드시 경기 종료 후 등록하세요.
- 점수 입력은 선택 사항이지만, 입력하면 상대방이 결과를 더 정확히 확인할 수 있습니다.
- 분쟁이 발생하면 대회 관리자에게 문의하여 해결하세요.
- 자동 확인은 상대방이 일정 기간 동안 응답하지 않을 경우 자동으로 처리됩니다.
