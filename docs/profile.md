# 프로필 및 계정 관리

계정 정보와 프로필을 확인하고 수정하는 방법을 안내합니다.

---

## MY 화면

1. 하단 탭(또는 좌측 메뉴)에서 **MY**를 선택합니다.

### 프로필 카드

- **이메일**: 로그인에 사용하는 이메일 주소
- **닉네임**: 다른 사용자에게 표시되는 이름

### 메뉴 항목

| 항목 | 설명 |
|------|------|
| **프로필 수정** | 닉네임, 표시 이름, 자기소개 변경 |
| **알림 설정** | 알림 설정 (준비 중) |
| **로그아웃** | 로그아웃 |

---

## 프로필 수정

1. MY 화면에서 **프로필 수정**을 클릭합니다.
2. 정보를 수정합니다.

| 항목 | 설명 |
|------|------|
| **닉네임 (Username)** | 필수, 다른 사용자에게 표시되는 고유 이름 |
| **표시 이름 (Display Name)** | 선택, 실제 이름이나 별명 |
| **자기소개 (Bio)** | 선택, 짧은 자기소개 |

3. **저장** 버튼을 클릭합니다.

> **팁:** 닉네임은 대회 참가자 목록, 경매 팀명, 트레이드 요청 등 플랫폼 전반에서 표시됩니다. 알아보기 쉬운 닉네임으로 설정하세요.

---

## 로그아웃

1. MY 화면에서 **로그아웃**을 클릭합니다.
2. 로그인 화면으로 이동합니다.

---

## 계정 관련 주의사항

- 이메일 주소는 변경할 수 없습니다. (회원가입 시 사용한 이메일 고정)
- 닉네임 변경은 즉시 적용되며, 대회 및 리그 전반에 반영됩니다.
- 비밀번호 변경 기능은 추후 지원 예정입니다. 비밀번호를 분실한 경우 관리자에게 문의하세요.
