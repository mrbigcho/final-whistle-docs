# 이적 시장

이적 시장은 경매 이후 팀 구성을 조정할 수 있는 공간입니다. 선수 방출, 팀 간 트레이드, 자유선수 입찰(FAAB)의 세 가지 방식을 지원합니다.

---

## 이적 시장 접근 방법

1. 리그 상세 화면에서 **이적 시장** 버튼을 클릭합니다.
2. 또는 직접 URL로 접속합니다.

---

## 탭 구성

이적 시장 화면은 세 개의 탭으로 구성됩니다.

| 탭 | 설명 |
|----|------|
| **선수 방출** | 내 팀 선수를 자유선수(FA)로 방출 |
| **트레이드** | 다른 팀과 선수 교환 |
| **자유선수 입찰 (FAAB)** | 방출된 자유선수를 예산으로 입찰 |

---

## 선수 방출

내 팀에서 원하지 않는 선수를 자유선수로 방출합니다.

1. **선수 방출 탭**을 선택합니다.
2. 내 팀 선수 목록이 표시됩니다.
3. 방출할 선수 옆의 **방출** 버튼을 클릭합니다.
4. 확인 대화상자에서 **확인**을 클릭합니다.

> **주의:** 방출된 선수는 자유선수(FA)가 되어 FAAB 입찰 대상이 됩니다. 방출 후에는 되돌릴 수 없습니다.

---

## 트레이드

다른 팀과 선수를 교환합니다.

### 트레이드 제안하기

1. **트레이드 탭**을 선택합니다.
2. **트레이드 제안** 버튼을 클릭합니다.
3. 내가 보낼 선수와 상대방에게 받을 선수를 선택합니다.
4. 상대 팀을 선택합니다.
5. **제안** 버튼을 클릭합니다.

### 트레이드 요청 처리

상대방이 트레이드를 제안하면 목록에 표시됩니다.

| 버튼 | 동작 |
|------|------|
| **수락** | 트레이드를 승인합니다. 즉시 선수가 교환됩니다. |
| **거절** | 트레이드를 거부합니다. |

### 트레이드 상태

| 상태 | 설명 |
|------|------|
| **대기 중 (Pending)** | 상대방이 아직 응답하지 않은 상태 |
| **수락됨 (Accepted)** | 트레이드가 성사된 상태 |
| **거절됨 (Rejected)** | 상대방이 거절한 상태 |
| **취소됨 (Cancelled)** | 제안자가 취소한 상태 |

---

## 자유선수 입찰 (FAAB)

방출된 선수나 경매에서 아무도 영입하지 않은 선수를 입찰로 획득합니다.

FAAB(Free Agent Acquisition Budget)는 자유선수 입찰에만 사용하는 별도 예산입니다.

### 입찰 방법

1. **자유선수 입찰 탭**을 선택합니다.
2. 자유선수 목록이 표시됩니다.
3. 원하는 선수 옆의 **입찰** 버튼을 클릭합니다.
4. **입찰 금액**을 입력합니다.
5. **입찰** 버튼을 클릭합니다.

### 입찰 마감 및 결과

- 각 선수에게는 **입찰 마감 시간**이 있습니다.
- 마감 시간이 지나면 가장 높은 금액을 제시한 팀이 선수를 획득합니다.
- 같은 금액일 경우 먼저 입찰한 팀이 우선순위를 가질 수 있습니다.

> **팁:** FAAB 예산은 시즌 내 계속 사용할 수 있습니다. 남은 예산을 효율적으로 관리하세요.

---

## 주의사항

- 트레이드는 상대방이 수락해야 성사됩니다.
- FAAB 입찰 금액은 내 남은 FAAB 예산을 초과할 수 없습니다.
- 로스터가 가득 찬 경우 선수를 추가로 영입하기 전에 방출이 필요합니다.
